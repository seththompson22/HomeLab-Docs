# HomeLab-Docs

A centralized knowledge base for my home lab and IT projects. This repository tracks hardware setups, installations, configurations, troubleshooting, scripts, and experiments. Each folder is organized by **hardware**, **projects/services**, or shared resources like scripts and diagrams.

---

## Overview

This repo is designed to document:

- **Hardware:** Physical servers, workstations, and networking devices, including BIOS, OS prep, and troubleshooting.
- **Projects / Installations:** Services, virtual machines, containers, or applications running on the lab hardware (e.g., Proxmox, Windows Server VMs, Minecraft server).
- **Scripts:** Automation, maintenance, or deployment scripts.
- **Diagrams:** Visual references for hardware layouts, network topology, and project architecture.
