# Dell PowerEdge R410 Setup

## Overview
This project documents the setup and operation of a Dell PowerEdge R410
used as an enterprise-style homelab server for learning hardware configuration,
RAID management, and virtualization. 

## Scope
Included:
- Hardware setup and firmware configuration
- RAID controller configuration
- Hypervisor preparation

Excluded:
- Guest OS configuration
- Application-level services

## Environment
- Server: Dell PowerEdge R410
- Management: iDRAC6
- Console: VGA
- Intended Hypervisor: Proxmox

## Documentation
All issues, failures, and troubleshooting steps are documented in
[INCIDENTS.md](./INCIDENTS.md).
